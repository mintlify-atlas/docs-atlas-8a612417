# Welcome to ZeroLimit

ZeroLimit is a **cross-platform desktop application** for monitoring AI coding assistant quotas using [CLIProxyAPI](https://github.com/router-for-me/CLIProxyAPI). Track your usage across multiple AI providers in one unified dashboard.

Built with **Tauri + React + Rust**, ZeroLimit runs seamlessly on **Windows**, **macOS**, and **Linux**.

## Supported AI Providers

ZeroLimit integrates with the following AI coding assistants through OAuth authentication:

| Provider | Auth Method | Support |
|----------|-------------|----------|
| Google Gemini | OAuth | Standard |
| Anthropic Claude | OAuth | Standard |
| OpenAI Codex | OAuth | Standard |
| Antigravity | OAuth | Standard |
| Kiro (CodeWhisperer) | OAuth | Plus Only |
| GitHub Copilot | OAuth | Plus Only |

<Note>
  **Plus version** provides additional support for GitHub Copilot and Kiro. Download from [CLIProxyAPI Plus Releases](https://github.com/router-for-me/CLIProxyAPIPlus/releases).
</Note>

## Key Features

### Comprehensive Quota Monitoring

ZeroLimit provides real-time visibility into your AI coding assistant usage:

- **Visual Progress Bars** - See quota consumption at a glance
- **Usage Statistics** - Track total requests, successful calls, and failures
- **Token Tracking** - Monitor cached tokens and reasoning tokens
- **Time-based Charts** - View usage trends by hour or by day
- **Model Breakdown** - Detailed statistics per AI model
- **API Analytics** - Request and token counts by provider

### Intelligent Dashboard

The dashboard page offers:

- Active account count display
- System connection status monitoring
- Request trend visualization with customizable time grouping
- Token usage trends with multi-line comparisons
- Expandable API details with model-level breakdowns
- Comprehensive model statistics table

### Flexible Configuration

<Tabs>
  <Tab title="Auto Download">
    Easiest setup method - ZeroLimit automatically downloads and configures CLI Proxy for you. Choose between Standard and Plus versions with a simple wizard.
  </Tab>
  <Tab title="Manual Setup">
    For users who prefer control - specify the path to your existing CLI Proxy executable and configure the management key manually.
  </Tab>
  <Tab title="Remote Server">
    Connect to a remote HTTPS server by providing the API base URL and management key. Perfect for team deployments.
  </Tab>
</Tabs>

## Technology Stack

- **Frontend**: React 19 with TypeScript
- **Desktop Framework**: Tauri 2.x
- **Backend**: Rust
- **UI Components**: Radix UI + Tailwind CSS
- **Charts**: Recharts
- **State Management**: Zustand
- **Internationalization**: i18next
- **Build Tool**: Vite
